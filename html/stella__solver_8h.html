enum stella_log_level

Stella solver log level.

Sets what the stella solver will log.

Enumerator
STELLA_LOG_STATUS 

log status messages

STELLA_LOG_PROBLEM 

Write the problem (matrix and rhs) to disk.

STELLA_LOG_RESIDUAL 

Write residual to stdout.

STELLA_LOG_VTK 

Write vtk files of computed solution.

STELLA_LOG_EIGS 

Output approximate eigenvalues from Krylov solver.
